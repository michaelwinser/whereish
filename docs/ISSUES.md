# Issues

Issue tracking has moved to GitHub Issues:

**https://github.com/michaelwinser/whereish/issues**

## Quick Reference

```bash
# List open issues
gh issue list

# Create a new issue
gh issue create

# View an issue
gh issue view <number>

# Close an issue
gh issue close <number>
```
